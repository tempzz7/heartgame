HEART Game Audio Files

This directory should contain the following audio files:
- heartbeat.wav: Sound of heartbeat
- hit.wav: Sound when player gets hit
- collect.wav: Sound when collecting a memory
- menu.wav: Sound when navigating menus
- attack.wav: Sound when attacking
- spare.wav: Sound when sparing
- menu_theme.mp3: Background music for menu
- game_theme.mp3: Background music for gameplay
- battle_theme.mp3: Background music for battles
- credits_theme.mp3: Background music for credits

You can create these using any sound editor or download free sounds from sources like:
- https://freesound.org/
- https://pixabay.com/sound-effects/
- https://opengameart.org/
